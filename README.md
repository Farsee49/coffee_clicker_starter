# Coffee Clicker DOM Project

### What needs to get done? 

Follow the directions in the code comments in the ```script.js``` file.  
You will need to:

- Make references to DOM nodes

- Complete the following functions:
  - ```updateCoffeeView```
  - ```clickCoffee```
  - ```unlockProducers```
  - ```getUnlockedProducers```
  - ```renderProducers```
  
 - Add eventListeners
